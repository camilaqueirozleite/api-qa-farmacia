# 📘 API QA Farmácia  
API desenvolvida para estudos e práticas de QA, incluindo autenticação, CRUD de medicamentos e registro de dispensações. Documentada com Swagger e estruturada em arquitetura MVC.

---

## 🚀 Tecnologias Utilizadas
- Node.js  
- Express  
- Swagger (OpenAPI 3.0)  
- Arquitetura MVC  
- JavaScript
